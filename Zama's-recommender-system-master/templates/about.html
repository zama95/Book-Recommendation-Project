About Project

Book Recommendation System

This Recommendation system is made by Zama Sayyed of MSc IT (SEM-) Seat No. 10313

The idea for taking this topic as my semester 1 project was something personal.
I personally find reading books as one of my favourite hobby. And making use of my hobby into my academic curriculum was the basis of me taking this topic as my project.
I was one hundred percent sure that i would learn and enjoy "MACHINE LEARNING" while working on this project.
The Model used are as follows:-
The HOME page which shows the Top 50 Books is made by using the popularity based recommendation system.
The RECOMMEND page is made by using Collaborative-Filtering.

The Model building was done using PYTHON on Jupyter Notebook.
And everything was then deployed/coded using PyCharm IDE
I have used Jinja2 template and FLASK for making this Recommendation system
